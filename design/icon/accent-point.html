修正 · 为什么不要对称点 ii

镜像点 = 等臂配重，它把构图重新配平回中心—— 等于亲手抹掉主星 偏居左下 的张力，整体塌回"绕中心对称 = 静止"。 视觉心理学：平衡 ≠ 对称；最稳又最活的是 非对称平衡。

所以把 accent 推到更靠右上角： 小质量 × 长力臂 去平衡近中心的大团——像 风铃 (mobile)，不是 天平。 |CA| = 0.363 > |CP| = 0.263，离心、有动势。

还有方向：螺线 向内 收敛到 P（起源 / 你从哪来）； 这点火花就该被 甩向外缘（右上 = 上扬、目标、"灵光"）。 内聚的大团 + 外抛的小火花 = 一个 矢量，不是一对静物。

修正 · 为什么十字要转向 iii

同朝向的缩小副本 = 克隆，读成"同一个东西重复两遍"， 挣不到 accent 的身份。副元素要靠 对比 立足。

深层是 两套坐标系：主星归 内部黄金引擎（对齐螺线的两条对角轴，倾斜 31.7° / 58.3°）；accent 归 外部画框（对齐 squircle 的边 → 立正正交十字）。

两套十字之间的夹角恰好 arctan(1/φ) ≈ 31.7°—— 还是黄金角。所以"不同"但不"随意"。 语义上：右上角一颗立正 ✦ = 通用的"闪光 / AI / 新"标记惯用位。

A 的几何锚点 iv

沿 轴 1（主星短臂所指的线）从中心 C 推向它与右边缘的交点 E = (1, 0.191)，取 黄金分割点： A = C + (E−C)/φ，即 CA : AE = φ : 1，得 A = (φ/2, 1/2φ) = (0.809, 0.309)。

A 仍在轴 1 上——主星的短臂直指它。 所以它是被引擎 甩出 的火花，而非凭空一点； 但它一到外缘就 转正、退到角落—— 这是 内部引擎向外部画框的交接。位置来自里，朝向来自外。
